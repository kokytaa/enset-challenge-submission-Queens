## 🚀 Overview

**PwnGPT** is an AI-powered assistant designed to help solve Capture The Flag (CTF) challenges.

This MVP focuses on leveraging **Generative AI (Google Gemini)** to analyze cybersecurity challenges and suggest structured solving strategies.

The goal is to provide a simple yet effective tool that assists users during CTF competitions.

---

## ✨ Features

- 🤖 **AI Assistant**
  Uses Google Gemini to analyze CTF challenges and suggest solutions.

- 🧠 **Basic Reasoning**
  Generates step-by-step approaches based on the challenge description.

- 📂 **File Inspection**
  Allows preview and basic analysis of provided challenge files.

- 📚 **Simple Knowledge Base (RAG)**
  Searches local files for relevant hints and references.

- 🌐 **Simple User Interface**
  Built with Streamlit for easy interaction.

---

## 🛠️ Tech Stack

- **Frontend**: Streamlit
- **Backend**: Python
- **LLM**: Google Gemini

## ⚙️ Installation

### Prerequisites

- Python 3.10+

### Setup

```bash
git clone https://github.com/kokytaa/PwnGPT.git
cd PwnGPT
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the application:

```bash
streamlit run app.py
```

Then:

1. Enter the challenge name
2. Provide the challenge description
3. Add optional hints
4. Let the AI suggest a solving strategy
